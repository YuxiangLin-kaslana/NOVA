# 字体资源

`make_weekly_figs.py` 和 `md_to_pdf.py` 需要一个同时含**中文+拉丁**的字体来渲染图表/PDF。
该字体(16MB)未入库,请按需下载到本目录:

```bash
curl -fsSL -o scripts/assets/NotoSansCJKsc-Regular.otf \
  https://github.com/notofonts/noto-cjk/raw/main/Sans/OTF/SimplifiedChinese/NotoSansCJKsc-Regular.otf
```
